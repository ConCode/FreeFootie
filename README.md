Prequisites:
* node (with npm)
* grunt-cli (npm package)
* ruby
* compass (ruby gem)

Starting work on this project is easy. You just need to install the prereqs for your chosen OS. After that, just follow the steps below:

    git clone https://github.com/ConCode/FreeFootie
    cd FreeFootie
    npm install
    bower install
    grunt server
